# liquid_glass_bottom_nav

A beautiful, liquid frosted-glass floating bottom navigation bar for Flutter.

## Features

- Frosted glass effect
- Liquid animation
- Highly customizable
- Minimal setup

## Getting started

Add `liquid_glass_bottom_nav` to your `pubspec.yaml`:

```yaml
dependencies:
  liquid_glass_bottom_nav: ^0.0.1
```

## Usage

```dart
import 'package:liquid_glass_bottom_nav/liquid_glass_bottom_nav.dart';

// Use LiquidGlassBottomNav as your bottom navigation bar
```
